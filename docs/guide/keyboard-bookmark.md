## 使用
1. 通过`鼠标左键`点击 or 按下电脑键盘上的`对应按键`可**直接**打开网站
2. `Shift + 按键` or `鼠标中键`点按书签以完成**后台**打开书签
3. `Alt + 按键`以完成**新标签页**打开书签
4. 还可通过设置`自定义快捷键`实现在浏览器内**全局**快速打开指定书签
